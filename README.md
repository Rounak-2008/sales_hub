# sales_hub
School Canteen Analytics Hub  A Python-based data analysis and visualization tool built for a Class 12 IP (Informatics Practices) project. It processes historical canteen sales data, generates graphical performance dashboards, and uses seasonal averaging to forecast future demand.

This repository hosts a comprehensive data analysis project developed in Python for the CBSE Class 12 Informatics Practices (065) curriculum.

Project Goal
The primary objective is to transform raw CSV sales data into actionable business intelligence for canteen management. The system focuses on minimizing inventory waste and maximizing service efficiency by predicting student demand.

Key Features
Data Processing: Utilizes Pandas for robust data loading, cleaning, and aggregation from a CSV source.

Performance Dashboards: Generates a 4-chart analytical dashboard using Matplotlib, showing:

Daily Sales Trends

Top Selling Food Items

Average Sales by Day of Week

Impact of Special Events (e.g., Sports Day)

Sales Forecasting: Implements a seasonal naive prediction model to forecast item-wise demand for the next 7 days, visualizing the required item mix.

Menu-Driven Interface: Provides a user-friendly console menu for easy report generation.

Technologies Used
Language: Python 3.x

Data Analysis: Pandas

Visualization: Matplotlib
